## How It Works

### Layer 1: Quantified Self (Automatic Data Collection)
```
Canvas Watcher Agent
  → Fetches assignments, grades, deadlines automatically
  → 100% data collection (no manual work)

Behavioral Signals Agent
  → Tracks typing speed, pause frequency, submission timing
  → Understands your work patterns

Emotional Signals Agent
  → Detects stress, confidence, motivation
  → Understands your emotional state

Biometric Signals Agent
  → Integrates Apple Health (sleep, heart rate, activity)
  → Understands your physical state

iOS Monitoring Agent
  → Uses Accessibility Framework
  → Records what you're doing on your phone
  → Tracks app usage and focus
```

### Layer 2: Second Brain (Knowledge Management)
```
Supabase Database (22 tables, fully normalized)
  → Persistent memory (never forgets)
  → Real-time subscriptions
  → Vector embeddings (semantic search)

Zettelkasten Automation
  → Breaks knowledge into atomic ideas
  → Creates semantic links automatically
  → Discovers patterns you missed
```

### Layer 3: Emotional Intelligence (Cognitive Prosthetic)
```
Emotional Continuity Agent
  → Remembers your family crisis
  → Remembers your stress patterns
  → Remembers your goals
  → Provides genuine support (not character)

Conversation History (Persistent)
  → Every conversation stored
  → Never forgotten
  → Searchable
  → Used for context
```

### Layer 4: Synthesis & Agency (Autonomous Action)
```
Situation Synthesizer
  → Understands your current context
  → Connects all signals (behavioral + emotional + academic)
  → Predicts your needs

Predictive Models
  → Predicts your grades
  → Detects when you'll struggle
  → Learns from outcomes

Ambient Recommendations
  → Gives you help WITHOUT you asking
  → Right advice at the right moment
  → Proactive (not reactive)

Autonomous Agency
  → Takes actions on your behalf
  → Schedules study sessions
  → Creates reminders
  → Learns what helps
```
