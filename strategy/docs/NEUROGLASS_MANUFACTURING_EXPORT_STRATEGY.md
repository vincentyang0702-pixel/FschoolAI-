## The Export Challenge: What You Need to Solve

### Challenge 1: US Import Regulations

**What you need:**
- FCC certification (for wireless devices)
- FDA compliance (if biometric sensors)
- RoHS compliance (environmental)
- CE marking (if selling in Europe)

**Timeline:**
- FCC: 3-6 months
- FDA: 2-4 months (if required)
- CE: 1-2 months
- Total: 6-12 months

**Cost:**
- FCC: $5,000-15,000
- FDA: $10,000-50,000 (if required)
- CE: $2,000-5,000
- Total: $17,000-70,000

---

### Challenge 2: Export from China

**What you need:**
- Export license from China (for certain electronics)
- Customs documentation
- Shipping logistics
- Tariffs and duties

**Timeline:**
- Export license: 2-4 weeks
- Customs: 1-2 weeks per shipment
- Shipping: 2-3 weeks by sea, 3-5 days by air

**Cost:**
- Export license: Free (but bureaucratic)
- Customs: Included in shipping
- Shipping: $0.50-2 per unit (by sea), $3-5 per unit (by air)
- Tariffs: 0-25% depending on product classification

---

### Challenge 3: Product Classification

**The key question:** What is NeuroGlass?

```
Option A: "Smart Card" (like crypto wallet)
├─ Lower tariffs (5-10%)
├─ Easier export
├─ But limited features

Option B: "Wearable Device" (like smartwatch)
├─ Higher tariffs (15-20%)
├─ More regulations
├─ But more features

Option C: "Audio Device" (like AirPods)
├─ Medium tariffs (10-15%)
├─ Standard regulations
├─ Good fit for NeuroGlass

Option D: "Medical Device" (if biometric)
├─ Highest tariffs (20-25%)
├─ Most regulations (FDA)
├─ Only if claiming health benefits
```

**Recommendation:** Classify as "Audio Device" (Option C)
- Fits the form factor
- Reasonable tariffs
- Standard regulations
- Easiest path


### Step 3: Set Up Import Process

**Process:**
```
1. Freight forwarder books shipment
2. Prepares customs documentation
3. Assigns HS code
4. Calculates duties
5. Shipment leaves China
6. Arrives at US port
7. US customs broker clears
8. You pay duties
9. Shipment released to warehouse
```

**Timeline:** 2-3 weeks by sea, 3-5 days by air
**Cost:** Duties are 10-15% of product value

---

### Step 4: Establish Customs Broker Relationship

**What they do:**
```
├─ File import documentation
├─ Calculate duties
├─ Arrange payment
├─ Clear customs
├─ Arrange delivery to warehouse
└─ Handle any issues

Cost: $500-1,000 per shipment (or flat fee)
```

**Recommendation:** Use established broker
- They know the process
- They have relationships with customs
- They can expedite if needed

## The Regulatory Checklist

### FCC Certification

```
Required if:
├─ Device has wireless (Bluetooth, WiFi)
├─ Device transmits RF signals
└─ Selling in US

Process:
1. Design must meet FCC standards
2. Send prototype to FCC lab
3. Lab tests RF emissions
4. Lab generates report
5. Submit to FCC
6. FCC issues certification

Timeline: 3-6 months
Cost: $5,000-15,000
```

**For NeuroGlass:** YES, required (has Bluetooth)

---

### FDA Compliance

```
Required if:
├─ Device claims health benefits
├─ Device is medical device
├─ Device treats/diagnoses/cures disease
└─ Device has biometric sensors (maybe)

For NeuroGlass:
├─ Recording audio: NOT medical
├─ Biometric sensors: Maybe (depends on use)
├─ Recommendation: Don't claim health benefits
└─ Result: FDA not required
```

**Action:** Don't claim health benefits initially
- Avoid FDA complexity
- Can add later if needed

---

### RoHS Compliance

```
Required if:
├─ Selling in EU/UK
├─ Device has electrical components
└─ Device has hazardous substances

Process:
1. Verify components are RoHS compliant
2. Get RoHS declaration from factory
3. Keep documentation
4. Display RoHS marking

Timeline: 1-2 weeks (mostly documentation)
Cost: $1,000-3,000
```

**For NeuroGlass:** YES, required if selling in EU

---

### CE Marking

```
Required if:
├─ Selling in EU
├─ Device has wireless
├─ Device is consumer electronics

Process:
1. Verify compliance with EU directives
2. Get CE declaration from factory
3. Keep technical documentation
4. Display CE marking

Timeline: 2-4 weeks
Cost: $2,000-5,000
```

**For NeuroGlass:** YES, required if selling in EU
